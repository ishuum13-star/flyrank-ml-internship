## 3. Three Target Tasks (for FL-02–FL-04)

### Task 1: Building/coding freelance client websites
**Success definition:** Site is fully deployed on Netlify, includes every page/feature 
the client requested, works correctly on mobile and desktop, and the client approves 
the first draft without requesting major revisions.

### Task 2: FlyRank ML weekly deliverables (notebooks + markdown submissions)
**Success definition:** Assignment meets 100% of the published "pass bar" criteria on 
the first submission — no revision request from the mentor — and I can explain the 
reasoning behind the work without re-reading my own notes.

### Task 3: Debugging code errors (Colab, JavaScript, or general coding)
**Success definition:** I can state the root cause of the error in one sentence 
(not just "AI fixed it"), and the same category of error doesn't recur in my next 
2-3 coding sessions.
